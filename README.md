## Информация о проекте

**Выполняет:** Звягинцев Александр, РИС-22-4  
**Тема проекта:** «Разработка управляющего портала для бизнес-процессов организации»

### Описание проекта

Данный проект направлен на разработку прототипа информационной системы, предназначенной для поддержки и управления бизнес-процессом рассмотрения обращений граждан в образовательной организации. В качестве объекта исследования выбран регламентированный процесс обработки обращений, используемый в Национальном исследовательском университете «Высшая школа экономики».

В рамках проекта предполагается провести анализ существующего процесса работы с обращениями граждан, выполнить его декомпозицию и построить модель текущего состояния процесса. На основе выявленных особенностей и проблемных этапов будет сформирована целевая модель процесса, предусматривающая использование специализированного программного портала.

Разрабатываемая система должна обеспечивать централизованную регистрацию обращений, распределение задач между ответственными исполнителями, контроль сроков их обработки, а также формирование и отправку официальных ответов гражданам. Предполагается, что внедрение подобного решения позволит повысить прозрачность процесса, улучшить контроль исполнения обращений и снизить нагрузку на сотрудников организации.
